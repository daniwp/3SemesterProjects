# angular-starter-2016
An starter project with Angular.JS

### Includes
- angular
- angular-ui-router
- angular-animate
- angular-toastr
- angular-bootstrap
- bootstrap-sass
- font-awesome

### Requirements:
- Gulp
- NPM

### How to run:

#### 1. Install node packages
`npm install`

#### 2. Run gulp.js
`gulp`
